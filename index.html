Quick Test Switcher

テストコードと実装コードを簡単に切り替えられるようにするためのVisual Studio用アドインです。 Quick JUnit Plugin for Eclipse のVisual Studio版が欲しかったので、作りました。

上記プラグインには、テストを実行するための機能も実装されていますが、 本アドインにはそれは実装されていません。 別のアドインとして、ビルド時にバックグラウンドでテストを実行する仕組みを提供する予定です。

用語

Quick JUnit Plugin for Eclipse
このアドインの元となったEclipse用のプラグインです。
テスティングペア
Quick JUnit Plugin for Eclipseのドキュメントから引用すると、
「テスティングペア」とは, 実装コードに対するテストコード, テストコードに対する実装コードのことを意味します。
です。 このアドインは、この「テスティングペア」を高速に切り替えるためだけに存在します。

使い方

対象のファイルを開いた状態で、Ctrlキーと0(ゼロ)を押すことで、対応するファイルが開きます。

このとき、エディタにフォーカスが当たっていなくても切り替えが行えます。

テスティングペア

現在のところ、テスティングペアを構成するためのルールは固定化されています。

  • 実装コードとテストコードのプロジェクトを分けること
  • テストコードのプロジェクト名は、実装コードのプロジェクト名の末尾に、以下のいずれかの語を付けること
    • Test
    • Tests
    • Scenario
    • Scenarios
    • Spec
    • Specs
    これらの前にドットが一つあっても構わない
  • テストコードのファイル名は、実装コードのファイル名の拡張子を除いた名前に、 以下のいずれかの語を付けること
    • Test
    • Scenario
    • Spec

このすべてを満たす実装コードとテストコードがテスティングペアとなります。
